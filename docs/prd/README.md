# PRDs

This directory stores product requirements documents (PRDs) that define PlotLot’s evolution into an agentic land-use and site-feasibility consultant harness.

Canonical working PRD for this branch lives in:

- `.omx/plans/prd-agentic-land-use-harness.md`

This directory exists so product/spec artifacts can be consumed outside the `.omx` planning system when needed.
