# MoonshotAI/kimi-cli

## What it is

- A terminal-first **coding/ops agent CLI**.
- Positions itself as an “AI agent that runs in the terminal” with code reading/editing, shell execution, and web search/fetch.

## Notable primitives

- **Shell command mode** inside the agent (explicit mode switch).
- **IDE integrations**:
  - VS Code extension
  - ACP (Agent Client Protocol) support (`kimi acp`).
- Explicit docs + zsh integration plugin.

## How it maps to PlotLot

- Useful reference for:
  - “agent as terminal shell” UX patterns
  - ACP-style integration as a generic seam for editor/IDE clients
  - keeping integrations explicit and opt-in rather than implicit tool soup
