# FrontierSWE — blog

## What it is

A project/blog describing **FrontierSWE**, an effort to benchmark coding agents on *ultra-long-horizon* challenges (harder, longer, and more realistic than many common benchmarks).

## Key ideas worth copying

- Existing public benchmarks often over-index on **short tasks** (small PR-scale changes, short runtimes).
- Long-horizon evaluation needs:
  - realistic environments
  - sustained progress over time
  - failure recovery and debugging behavior

## How it maps to PlotLot

- Strengthens the case for PlotLot harness evals that test:
  - multi-step feasibility workflows
  - tool correctness under retries/timeouts
  - evidence completeness + provenance
  - long conversational sessions without drift
