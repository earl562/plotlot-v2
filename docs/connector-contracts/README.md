# Connector Contracts

This directory stores normalized contracts for external systems such as:

- Municode / ordinance sources
- ArcGIS Hub / FeatureServer / MapServer
- Socrata/Tyler SODA open-data portals
- Google Workspace and CRM connectors

Contracts specify:

- request/response shapes
- provenance/citation requirements
- freshness and licensing expectations
- governance risk class and approval requirements
- failure modes

- `municode.md`
- `opendata-arcgis-socrata.md`
- `workspace-connectors.md`
